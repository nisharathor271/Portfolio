# Nisha Rathore - Portfolio Website

A modern, responsive portfolio website built with React and Tailwind CSS. This portfolio is designed to be mobile-friendly and works seamlessly across iOS, Android, and desktop devices.

## Features

- **Responsive Design**: Optimized for all screen sizes (mobile, tablet, desktop)
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Mobile-First**: iOS and Android friendly with touch-optimized interactions
- **Fast Loading**: Optimized performance and lightweight components
- **Accessible**: Built with accessibility best practices
- **SEO Ready**: Structured for search engine optimization

## Sections

- **Hero**: Eye-catching introduction with call-to-action buttons
- **About**: Professional background and highlights
- **Experience**: Timeline of work experience
- **Skills**: Technical and soft skills with progress indicators
- **Projects**: Showcase of featured work and projects
- **Contact**: Contact form and social media links

## Tech Stack

- **React 18**: Modern React with hooks and functional components
- **Tailwind CSS**: Utility-first CSS framework for rapid styling
- **Lucide React**: Beautiful, customizable icons
- **Responsive Design**: Mobile-first approach with breakpoints

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. Clone or download the project files
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

### Building for Production

To create a production build:

```bash
npm run build
```

This creates a `build` folder with optimized files ready for deployment.

## Customization

### Personal Information

Update the following files with actual content:

1. **src/components/Hero.js** - Name, title, and introduction
2. **src/components/About.js** - Professional background and story
3. **src/components/Experience.js** - Work experience and achievements
4. **src/components/Skills.js** - Technical and soft skills
5. **src/components/Projects.js** - Portfolio projects and work samples
6. **src/components/Contact.js** - Contact information and social links

### Styling

- **Colors**: Modify `tailwind.config.js` to change the color scheme
- **Fonts**: Update font imports in `public/index.html` and `tailwind.config.js`
- **Layout**: Adjust spacing and layout in individual components

### Images

Replace placeholder images in the Projects section with actual project screenshots. You can use services like:
- Unsplash for stock photos
- Your own project screenshots
- Custom graphics and illustrations
